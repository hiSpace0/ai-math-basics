# 人工智能数学基础

本项目是《人工智能数学基础》课程的学习笔记与代码实践，主要以 Jupyter Notebook 形式呈现各章节核心数学概念的可视化与编程实现。

## 项目结构

```
├── 人工智能数学基础.pdf          # 课程教材
├── 第2章 初等数学/               # 激活函数、函数对比等
├── 第3章 微积分初步/             # 微积分基础概念与实现
├── 第4章 线性代数/               # 主成分分析、协方差等
├── 第5章 概率论/                 # 泊松分布等概率模型
├── 第6章 数理统计/               # 贝叶斯推断、回归分析、蒙特卡洛等
├── 第7章 函数逼近/               # 函数逼近方法
├── 第8章 最优化理论/             # （此前课程已学，未补充）
├── 第9章 信息论/                 # （此前课程已学，未补充）
└── 第10章 图论/                  # （此前课程已学，未补充）
```

> **说明**：第8章（最优化理论）、第9章（信息论）、第10章（图论）由于在此前课程中已经学过，本项目未重复编写相关内容。

## 各章节内容

| 章节 | 内容概要 |
|------|----------|
| 第2章 初等数学 | 激活函数（Sigmoid、ReLU、Tanh 等）对比与可视化 |
| 第3章 微积分初步 | 导数、梯度等微积分基础概念 |
| 第4章 线性代数 | 主成分分析（PCA）、协方差椭圆等 |
| 第5章 概率论 | 泊松分布等概率分布 |
| 第6章 数理统计 | 贝叶斯推断、样本标准差可视化、配对检验、蒙特卡洛模拟、回归分析 |
| 第7章 函数逼近 | 函数逼近方法 |

## 环境依赖

本项目需要以下 Python 包：

| 包名 | 版本 |
|------|------|
| numpy | 1.24.3 |
| matplotlib | 3.10.9 |
| sympy | 1.14.0 |
| scipy | 1.15.2 |

安装命令：

```bash
conda activate env && pip install numpy==1.24.3 matplotlib==3.10.9 sympy==1.14.0 scipy==1.15.2
```

## 使用方式

使用 Jupyter Lab 打开各章节的 `.ipynb` 文件即可查看和运行代码：

```bash
conda activate env && python -m jupyter lab
```
